Git

Git ja Github on erinevad.

Git on kohalik rakendus, ehk vaid kasutaja seadmes ning ei ole otseselt võrguga seotud.
Github on veebipõhine lehekülg, kuhu saab üles laadida koodi, et pääseda sellele ligi teistest seadmetest.

Terminid

  • Repositories/Repositoorium: Koht kus koodiajalugu on salvestatud.
  • Branches/Harud: Repositooriumi sees, nagu “kaust” mis sisaldab commit’e.
  • Commits: Erinevad etapid/versioonid koodist.

Command

$ git init

Initializes a local Git repository
